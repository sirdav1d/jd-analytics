3. Executar o plano aprovado e, ao concluir, seguir para o passo 4.
4. Marcar a tarefa como concluída e, abaixo da tarefa, registrar um resumo do que foi feito.
5. Em qualquer etapa que exija referência externa, usar o Context7 MCP para buscar documentação atualizada.
